# Mtrack
L’objectif de ce projet est de réaliser un outils d’aide à la programmation permettant d’analyser la gestion de la mémoire dans un code complet (éventuellement modulaire, mais exécutable). Cette bibliothèque va agir comme un «patch» en détournant les fonctions de gestion de mémoire vers des clones permettant la récolte d’informations à exécution.
